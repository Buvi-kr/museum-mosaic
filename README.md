# 🔭 포천아트밸리 천문과학관 포토 모자이크 시스템 (V5)

본 프로젝트는 포천아트밸리 천문과학관에서 운영되는 **로컬 독립 구동형 실시간 포토 모자이크 전시 플랫폼**입니다.
기존의 고정형 디스플레이 방식을 탈피하여, 큐레이터(관리자)가 전시 상황에 맞춰 실시간으로 모자이크 조각 수와 배경을 통제할 수 있는 **V5 통합 관리자 아키텍처**가 적용되어 있습니다.

---

## 🌟 주요 기능 (Features)

### 1. 실시간 모바일 참여 (Live Mobile Participation)
- 관람객이 전시장에 띄워진 QR 코드를 스캔하여 사진을 업로드하면, 대형 디스플레이에 화려한 우주적 이펙트(Stellar Pop)와 함께 즉시 사진 타일이 날아와 박힙니다.
- Cloudflare Tunnel 자동 연동을 통해 별도의 외부 서버 비용이나 고정 IP 없이도, 무선 인터넷(LTE/5G)이 가능한 환경이면 어디서든 외부 접속을 지원합니다.

### 2. 통합 관리자 대시보드 (CMS Admin Panel)
- **접속 주소:** `http://localhost:3000/admin`
- **템플릿 즉각 전환:** 버튼 한 번으로 모자이크 격자의 밀도를 4단계로 자유자재로 스위칭합니다. (6칸, 12칸, 20칸, 28칸 지원)
- **동적 튜닝 노브:** 모자이크 완성 시 몇 칸을 리셋하고 몇 칸을 유지할지, 팝업 세리머니는 몇 초간 보여줄지 관리자가 실시간으로 조율할 수 있습니다.

### 3. 지능형 배경 자동 슬라이싱 (Auto-Slicing Engine)
- 관리자 페이지에서 새로운 우주(또는 특별 행사용) 배경 이미지를 업로드하면, 서버 내부의 `sharp` 엔진이 즉시 1920x1080으로 리사이징한 후 현재 설정된 격자 템플릿 갯수(ex: 6칸, 14칸 등)에 맞춰 **자동으로 배경 조각을 재단(Slicing)하여 교체**합니다.
- 기존에 쓰이던 배경 사진과 조각 파편들은 영구 유실되지 않고 `history/bg_history/[날짜시간]` 폴더로 안전하게 아카이빙(보관) 됩니다.

### 4. 무한 사이클 리셋 및 영구 보관 로직
- 모자이크(예: 14칸)가 100% 가득 차면 전체 화면에 축하 이펙트가 발생합니다.
- 동시에 백그라운드에서는 14개의 각기둥 사진들을 원본 화질로 병합하여 1장의 1920x1080 완성본 액자로 만들어 `history/completions/`에 영구 저장합니다.
- 이후 설정된 대기 시간(예: 5초)이 지나면 지정된 개수(예: 절반)만큼만 랜덤으로 남기고 나머지는 비워내어 다음 관람객이 이어서 참여할 수 있도록 **부분 리셋(Partial Reset)**을 단행합니다.

## ⚙️ 사전 요구 사항 (Prerequisites)

시스템을 실행하기 전에 로컬 PC에 다음 프로그램들이 설치되어 있어야 합니다:
1. **Node.js** (LTS 버전 권장, v18 이상)
2. **구글 크롬(Chrome) 브라우저** (전시용 Kiosk 모드 구동용)

*※ 외부망 연결을 위한 Cloudflare 터널링용 실행 파일(`cloudflared.exe`)은 프로젝트 폴더 내부에 로컬로 내장되어 있으므로, 별도로 설치하실 필요가 없습니다.*

## 🚀 설치 및 실행 방법 (How to Run)

### 1. 패키지 설치
최초 1회만 수행합니다. 프로젝트 폴더 최상단에서 명령 프롬프트(CMD) 또는 터미널을 열고 아래 명령어를 입력합니다:
```bash
npm install
```

### 2. 서버 및 디스플레이 구동
설치가 완료되었다면, 더 이상 터미널을 열 필요 없이 폴더 최상단에 있는 **`start.bat`** 파일을 더블 클릭합니다.
- 서버가 켜지면서 내부적으로 `cloudflared` 임시 터널이 자동 구동됩니다.
- 검은색 터미널 창에 QR 코드 접속을 위한 `https://xxxx.trycloudflare.com` 주소가 자동으로 발급됩니다.

### 3. 크롬 키오스크(Kiosk) 모드 및 듀얼 모니터 세팅 (Tip)
전시장에 1번 모니터(안내용)와 2번 모니터(대형 디스플레이)가 연결되어 있을 경우, 크롬의 키오스크 모드를 활용해 전체화면으로 고정할 수 있습니다.
명령 프롬프트에서 모니터 위치(`--window-position=X,Y`)를 지정하여 실행하세요:
```bat
# 1번 모니터 (좌측) - 가이드 화면
start chrome --kiosk --app=http://localhost:3000/guide.html --window-position=0,0

# 2번 모니터 (우측, 1번 모니터 해상도가 1920x1080일 경우) - 디스플레이 화면
start chrome --kiosk --app=http://localhost:3000/display.html --window-position=1920,0
```

## 🔧 관리자 모드 가이드 (Admin Guide)

관리자 페이지 접속: `http://localhost:3000/admin`

1. **그리드 템플릿 변경**: '6칸', '12칸', '20칸', '28칸' 중 하나를 선택하고 [저장]을 누르세요. 저장 즉시 `display.html`이 스스로 새로고침되며 칸 수가 변경됩니다. (기존 조각들은 백그라운드에서 다시 자동 재단됩니다.)
2. **유지할 칸 수 (Keep Count)**: 관람객 사진이 다 찼을 때 완전히 비울지(0), 아니면 몇 개를 살려두고 이어서 진행할지 조절합니다.
3. **새 배경 업로드**: 특별한 날에 맞춘 우주 배경 `.jpg` 또는 `.png`를 업로드하세요. 서버가 알아서 배경을 적용하고, 현재 칸 수에 맞춰 수십 개의 퍼즐 조각으로 쪼개줍니다.

- 모자이크 렌더링 방식 최적화: CSS 블렌드(blend) 대신, 백엔드 `sharp` 합성 엔진을 통해 서버에서 이미 합성 완료된 사진을 화면에 부드럽게 띄우는 방식으로 부하 최소화 및 호환성 확보.

**2. 모자이크 격자 상태 (Grid Layout) 다각화**
- 현장 분위기와 인구 밀집도에 맞춰 큐레이터가 상황을 자유롭게 통제할 수 있도록 **[6칸, 12칸, 20칸, 28칸]**의 총 4가지 전용 하드코딩 템플릿(Grid Template) 라인업 구축 완료 (`src/gridTemplates.js`).
  - **6칸(3x2)**: 관람객 한 명 한 명의 얼굴이 거대하게 노출되는 초고속 회전율 모드
  - **12칸(4x3)**: 적절한 크기와 빠른 회전율을 지닌 캐주얼 모드
  - **20칸(5x4)**: 군중과 개인의 조화가 적절한 황금비율 밀도 모드
  - **28칸(7x4)**: 수많은 사람이 한 화면에 꽉 들어차는 대규모 군중용 파노라마 모드

**3. 무한 사이클(Loop) & 부분 리셋 (Partial Reset) 알고리즘 도입**
- 화면이 100% 꽉 차버린 직후 전부 하얗게 초기화되어 관람객들의 허무함을 유발하던 문제 해결.
- **부분 리셋 로직**: 100% 완성 시 화려한 세리머니 이펙트 후 지정된 대기시간(초)이 지나면, 사전에 세팅한 특정 갯수(예: 절반)만큼만 무작위로 남기고 나머지만 부드럽게 지워내어 다음 관람객이 자연스럽게 이어갈 수 있도록 설계.
- **고해상도 영구 아카이빙**: 화면이 100% 채워지는 영광의 순간마다, 백그라운드 서버 엔진(`sharp`)이 즉시 가동되어 슬롯들의 원본을 병합하여 1장의 1920x1080 "오늘의 엑스포 샷"을 `/history/completions/` 에 영구 저장.

**4. V5 통합 관리자 대시보드 (`/admin`) 및 자동화 엔진 탑재**
- **런타임 Config DB화**: 서버를 껐다 켤 필요 없이 작동 중에 설정을 바꿀 수 있도록 `config.json` 도입.
- **프라이빗 대시보드 (`http://localhost:3000/admin`) 신설**:
  - 관리자가 즉시 **격자 템플릿 변경 (6/12/20/28칸)**, **유지할 칸 수 (Keep Count)**, **세리머니 대기시간**을 슬라이더와 셀렉트박스로 0.1초 만에 조종 가능.
- **오토 슬라이싱 (Auto-slicing) 엔진**: 
  - 과거 터미널(cmd) 창에서 `node slice_bg.js`를 따로 치던 번거로움 완전 제거. 관리자 대시보드에서 새로운 `.jpg/.png` 우주 배경을 업로드하면 서버가 이를 감지해 **즉석에서 1920x1080 해상도 정규화 후 현재 활성화된 템플릿(ex: 20칸)에 맞게 수십 조각으로 자동 재단 및 배포.**
- **동적 실시간 동기화 (Real-Time Sync)**: 관리자가 설정을 저장하거나 배경을 바꾸는 즉시, `Socket.io`의 `config_updated` 이벤트가 발송되어 전시장 메인 디스플레이(`display.html`)와 안내 모니터(`guide.html`)가 스스로 새로고침하며 즉각 새로운 모습으로 갱신됨.
- **히스토리 자동 보존**: 배경이나 템플릿 변경 시 이전까지 쓰던 배경 이미지와 조각 파편들은 영구 삭제되지 않고 `/history/bg_history/` 경로로 타임스탬프와 함께 자동 압축 보관 처리되어 전시 기록 유실을 방지.
